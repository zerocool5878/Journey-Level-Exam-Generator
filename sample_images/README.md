### Recommended Image Formats

- **Width:** 300-800 pixels (optimal for PDF layout)
- **File Size:** Under 1MB per image for best performance
- **Resolution:** 150-300 DPI for print quality

## üí° Creating Your Own Images

### Best Practices
1. **High Contrast** - Ensure text is readable
2. **Simple Layout** - Avoid cluttered images
3. **Consistent Style** - Match your organization's standards
4. **Proper Labels** - Include clear descriptions
5. **Copyright Safe** - Only use images you have rights to use

### Tools for Creating Images
- **Free:** GIMP, Paint.NET, Canva
- **Professional:** Adobe Photoshop, Illustrator
- **Diagrams:** Draw.io, Lucidchart, Visio
- **Screenshots:** Snipping Tool, Greenshot
